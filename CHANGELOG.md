### 🤖 AI 功能增强

#### 1. 智能总结"重新生成"接入豆包大模型
- 原来点击"重新生成"只有一个旋转动画，没有真正调用 AI
- 现在点击后调用豆包文本接口，将当日记忆卡片标题作为上下文，生成新的中文学习总结
- 按钮生成中显示"生成中..."并持续旋转，完成后自动更新文案

#### 2. 批注分析新增"内容概述"模块（约300字）
- 优化 AI 分析 Prompt，要求模型按两个层次输出：
  - `overview`：对整张截图内容的全局概述（约300字）
  - `detailIntro` + `detailSections`：专注于用户圈选/批注部分的重点解析
- 右侧详情抽屉新增两个带色标区块：
  - 蓝色「内容概述」：展示全局300字概述
  - 红色「圈选重点」：展示具体批注分析

#### 3. API 配置完全内置，用户不可见
- 删除侧边栏底部的 API Key 配置面板
- 删除批注 Modal 左上角的"AI 已配置 / 配置 AI"按钮及弹出面板
- API Key 与 Model ID 仅从环境变量读取，学生打开即用

### 🎨 UI / 交互优化

#### 11. 记忆详情抽屉图片显示真实比例
- 原来：`maxHeight: 260px` + `objectFit: cover`，图片上下截断
- 现在：`height: auto`，按图片原始宽高比完整显示

#### 12. 侧边栏宽度改为自适应
- 原来：固定 `w-96`（384px）
- 现在：`clamp(200px, 22%, 320px)`，响应 iPad 竖/横屏切换
  - iPad 竖屏（768px）→ 200px
  - iPad 横屏（1024px）→ 225px
  - 桌面（1440px+）→ 320px

#### 13. 批注完成后 Toast 提示样式升级
- 原来：绿色 ✅ emoji + "已同步至记忆"
- 现在：深色胶囊 + 渐变星形图标 + "已同步至AI记忆" + 蓝色"查看"文字

#### 14. 左下角缩略图调整
- 尺寸：固定 140×90px → 响应式 `18vw`（约160~260px），高度跟随真实比例
- 间距：距左/下 16px → **60px**
- Toast 图标：18px → 14px，更轻盈
